# Function to get robots.txt is structured form. Memoised

Function to get robots.txt is structured form. Memoised

## Usage

``` r
polite_fetch_rtxt(..., user_agent, delay, verbose)
```

## Arguments

- ...:

  arguments passed to
  [`robotstxt::robotstxt()`](https://docs.ropensci.org/robotstxt/reference/robotstxt.html)

- user_agent:

  user agent string

- delay:

  default delay

- verbose:

  logical
