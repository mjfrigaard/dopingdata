# Clean/tidy USADA sports

Clean/tidy USADA sports

## Usage

``` r
clean_sports(df, sport_col, tidy = TRUE)
```

## Arguments

- df:

  data.frame/tibble of processed USADA dataset with messy sports

- sport_col:

  character sport column (usually `sport`)

- tidy:

  logical. Tidy the sports after cleaning/wrangling?

## Value

tibble with cleaned sports
