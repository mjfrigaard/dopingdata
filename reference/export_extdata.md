# Export data to `inst/extdata/` or `inst/extdata/raw`

Export data to `inst/extdata/` or `inst/extdata/raw`

## Usage

``` r
export_extdata(x, path = "", type = "csv")
```

## Arguments

- x:

  data to export

- path:

  path to 'inst/' folder (and only 'inst/' folder!)

- type:

  string, type of exported file (`"csv"`, `"rds"`, or `"tsv"`)

## Value

exported data
