# ggplot2 theme (doping data)

ggplot2 theme (doping data)

## Usage

``` r
theme_ggp2g(
  base_size = 11,
  base_family = "Ubuntu",
  base_line_size = base_size/22,
  base_rect_size = base_size/22
)
```

## Arguments

- base_size:

  size of font

- base_family:

  family of font

- base_line_size:

  size of lines in base graph

- base_rect_size:

  size of rectangle in base graph

## Value

ggplot object
